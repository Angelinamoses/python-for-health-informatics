# Contributing to Python for Health Informatics

Thank you for your interest in contributing! This document provides guidelines for contributing to the project.

## Code of Conduct

Please be respectful, inclusive, and constructive in all interactions.

## How to Contribute

### Reporting Bugs
1. Check if the issue already exists in [Issues](https://github.com/Angelinamoses/python-for-health-informatics/issues)
2. If not, create a new issue with:
   - Clear title describing the bug
   - Steps to reproduce
   - Expected vs. actual behavior
   - Python version and environment details

### Suggesting Enhancements
1. Open a new issue with the `enhancement` label
2. Clearly describe the feature and use case
3. Explain how it aligns with the health informatics curriculum
